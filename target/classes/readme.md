Para poner en marcha la aplicación, puede utilizar el comando de Maven o el wrapper de Maven.

1. Uso de Maven (si está instalado)
Ejecute el siguiente comando en la raíz del proyecto:

mvn spring-boot:run

2. Uso del Wrapper de Maven (si Maven no está instalado globalmente)
Ejecute el siguiente comando en la raíz del proyecto:

./mvnw spring-boot:run

Acceso a la Aplicación
Una vez que la aplicación se haya iniciado correctamente, puede acceder a ella a través de la siguiente URL en su navegador:

http://localhost:8081/login
